# Library Management System (MERN Stack)

## Description
A full-stack library management system built using MongoDB, Express, React, and Node.js.

## Features
- Add, update, delete books
- Issue and return books
- Manage users
- Responsive UI

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## How to Run
### Backend
cd library-server
npm install
npm start


### Frontend


cd frontend
npm install
npm start
